Settings - General

All settings to modify Axis3's general behavior can be changed here.

Changes are automatically saved by profile.

Basic Axis Settings

•Allow close from system menu - This allows you to close Axis3 with the red box with the X on the top right corner. *On by default.
*Note* If off, you must close it with the Exit/Close Axis3 menu option on the right-most side.
•Always on top - This allows to keep Axis the top most window on your desktop. *Off by default.

•Command Prefix - You can specify a different command prefix using this option. *Dot (.) by default.
•Interface Language - Here you can select the language Axis3 will use for the interface. *Default is English and only one provided for now.

*Note* It is possible to create a new language database using the DataEdit/Language menu available in DevMode.

 

Path Settings
•Default Ultima Online Client - This is the path to your Ultima Online Client. It is used by the launcher to start UO *Default is set the first time Axis3 is launched.
•Custom Client Title - Here you can set a custom client window title in case Axis3 can’t find the UO client window. ‘Ultima Online’ and ‘UOSA’ are the values used internally. *Default is blank.

•Default Mulfile Path - This is the default path to your Ultima Online .mul/.uop folder (normally same as client) *Default is set the first time Axis3 is launched.

•Use same path as default client – Setting this will make the Mulfile Path the same as the client path. *On by default

Reset settings
This will remove all saved profile settings for the currently loaded profile, forcing Axis3 to use the default settings.
